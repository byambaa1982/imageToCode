# Static Assets Directory

This directory contains static assets for the Screenshot to Code application.

## Structure

- `images/` - Images, icons, and visual assets
- `css/` - Stylesheets
- `js/` - JavaScript files
- `fonts/` - Custom fonts (if any)

## Required Assets

The following assets should be added to complete the frontend setup:

### Favicon Files
- `favicon.ico` - Main favicon
- `apple-touch-icon.png` - Apple touch icon (180x180)
- `favicon-32x32.png` - 32x32 favicon
- `favicon-16x16.png` - 16x16 favicon

### Open Graph Image
- `images/og-image.jpg` - Social media sharing image (1200x630)

### Logo Assets
- `images/logo.svg` - Main logo (SVG format)
- `images/logo-white.svg` - White version for dark backgrounds
- `images/logo-icon.svg` - Icon-only version

### Demo Images
- `images/demo-screenshot.png` - Example screenshot for demos
- `images/demo-code.png` - Example generated code
- `images/hero-bg.jpg` - Hero section background (optional)

## Optimization Notes

- Use WebP format for images when possible
- Optimize images for web (compress and resize appropriately)
- Use SVG for icons and logos
- Implement lazy loading for non-critical images
