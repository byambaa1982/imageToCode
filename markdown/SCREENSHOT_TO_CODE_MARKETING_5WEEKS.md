def extract_code_blocks(response):

    css = re.search(r'```css\n(.*?)\n```', response, re.DOTALL)
    js = re.search(r'```javascript\n(.*?)\n```', response, re.DOTALL)
    return {'html': html, 'css': css, 'js': js}
```

**Solution**: Switched to Claude 3.5 Sonnet for lower priority conversions 
(60% cheaper, 85% as accurate).

**Challenge 3: Complex Layouts**
Multi-column dashboards sometimes got positioning wrong.

**Solution**: Added specific Flexbox/Grid instructions to prompt. Still 
not perfect, but 80% success rate now.
