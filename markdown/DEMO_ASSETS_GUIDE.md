- Upload screenshot (drag & drop animation)
- Framework selection (React highlighted)
- Processing animation (progress bar, 30-45 seconds)
- Generated code reveal
- Download button click

**60-90 seconds: Results & CTA**
- Side-by-side: original design vs. live code
