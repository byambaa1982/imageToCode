## Uploading Screenshots

### Supported Formats

- **PNG**: Best for screenshots with transparency
- **JPG/JPEG**: Good for general screenshots
- **WebP**: Modern format with good compression

### File Requirements

- **Maximum Size**: 10MB per image
- **Resolution**: Higher resolution images (1920x1080+) work best
- **Quality**: Clear, well-lit screenshots produce better results

### Upload Tips

✅ **Do:**
- Use high-resolution screenshots
- Ensure good contrast and clarity
- Include complete UI sections
- Crop out unnecessary browser chrome

❌ **Avoid:**
- Blurry or low-resolution images
- Screenshots with overlays or annotations
- Images with watermarks or copyright text
- Partial or cut-off UI elements

## Troubleshooting

### Common Issues

#### "Image Upload Failed"
- Check file size (max 10MB)
- Verify file format (PNG, JPG, WebP)
- Try a different browser
- Clear browser cache

#### "Conversion Taking Too Long"
- Normal processing: 30-60 seconds
- High traffic periods may be slower
- Check your internet connection
- Refresh page if stuck

#### "Generated Code Doesn't Match Screenshot"
- Try a clearer, higher-resolution image
- Ensure good contrast in the screenshot
- Crop out irrelevant parts
- Contact support with the image

#### "Low Credits Warning"
- Purchase more credits from pricing page
- Check conversion history for usage
- Consider bulk package for better value

## Best Practices

### Screenshot Quality

1. **High Resolution**: Use 1920x1080 or higher
2. **Good Lighting**: Avoid dark or poorly lit screens
3. **Full Components**: Include complete UI sections
4. **Clean Background**: Solid colors work best

### Framework Selection

1. **Match Your Project**: Choose framework you're actually using
2. **Consider Team**: Pick what your team knows
3. **Project Size**: React/Vue for apps, HTML for simple sites

### Code Usage

1. **Review Output**: Always review generated code
2. **Test Responsiveness**: Check mobile compatibility
3. **Customize**: Adapt to your specific needs
4. **Version Control**: Commit to Git for tracking

### Optimization Tips

1. **Batch Similar Screens**: Convert related UI at once
2. **Start Simple**: Begin with basic layouts
3. **Iterate**: Use feedback to improve results
4. **Save Favorites**: Bookmark successful conversions
